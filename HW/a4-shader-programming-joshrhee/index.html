Shaders, shaders everywhere!
Pick a shader effect:

1. Checkerboard: First, app.ts will pass the input slider's value. To convert the number of squares within the [u,v] range of [0...1,0...1] to [5...20,5...20], we use formula "numberOfSquares" = floor(inputSliderValue * 15 + 5). Once we get the number of squares, we subtract x coordiate by y coordinate like "xSubY" = floor(v_texcoord.x * numOfSquares) - floor(v_texcoord.y * numOfSquares) and check that total value is even number or odd number. If the xSubY is even number, then we color the Checkerboard black, otherwise white, so we can color black and white in zigzag pattern.

2. Swiss Chees: First, app.ts will pass the input slider's value. Then, we use the slider value times 0.2 and use this value for our circle's radius. Once we get the circle's radius, we set some circles' center position. I set up 9 circle's center position. Then check if the distance between circle's center position and texcoord, and if the distance is less than the circle's radius, this means our texcoord is inside of the circle, so we can discard it, so we can draw circles.

3. Orbit: First, app.ts will pass the time value. Then, we edit the position = vec3(position.x + cos(time) * radius, position.y + sin(time) * radius, position.z), so my object is moving a circle shape. I set up radius = 500, so my object makes a big circle shape.
